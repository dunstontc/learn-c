# Types


## Primitive Types
- `int` 
  - Integer; a whole number.
- `double` 
  - A double-precision floating point value.
- `char` 
  - A single character.
- `void` 
  - Valueless special purpose type. 

- `float` 
  - Floating point value; ie a number with a fractional part.
- `short`
- `long`
- `long long`
